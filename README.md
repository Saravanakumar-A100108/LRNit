# LRNit

Learn. Build. Lead.

Empowering the next generation of tech leaders through immersive learning, real-world building, and transformative leadership.

## 🚀 Getting Started

### Prerequisites
- Node.js & npm installed

### Setup
```sh
# 1. Clone the repository
git clone https://github.com/Saravanakumar-A100108/LRNit.Org.git

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```

## 🛠️ Technologies
- **Vite** (Frontend Tooling)
- **TypeScript** (Language)
- **React** (Framework)
- **Tailwind CSS** (Styling)
- **Shadcn UI** (Component Library)
- **Lucide React** (Icons)
- **Framer Motion** (Animations)
- **Supabase** (Backend & Database)

## 📁 Project Structure
- `src/components/`: Reusable UI components
- `src/pages/`: Main page components (Home, Team, Admin, etc.)
- `src/integrations/`: Supabase client configuration
- `supabase/`: Database migrations and Edge Functions
